About this tool. ATF header parsing is reconstructed from real sample files rather than official docs, so treat unfamiliar files as best-effort. Verified pixel decoding: raw RGBA/BGRA/RGB/BGR, DXT1/DXT5 (S3TC), and ETC1 — including the dual-plane ETC1 "color + alpha-as-grayscale" convention used by these particular files. PVRTC/ETC2/ASTC are detected but not decoded yet. .atf.lzma files are decompressed automatically (classic LZMA "alone" container) before parsing. Atlas XML support covers the Starling/Sparrow/TexturePacker <SubTexture> format. If anything parses wrong, every value in the Preview panel is manually editable.
